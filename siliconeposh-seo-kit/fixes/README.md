## 🧭 What is Silicon Epoch?

We have entered the **Silicon Epoch** — the decade in which artificial intelligence rewrites science, medicine, work, geopolitics, and the texture of daily life.

**Silicon Epoch** is a citation-driven, single-page-per-topic field guide tracking:

- 🧬 **Every frontier lab** — OpenAI, Anthropic, Google DeepMind, xAI, Meta, Mistral, DeepSeek, Qwen, Moonshot, Cohere, Zhipu, ByteDance, Baidu, Tencent
- 🧠 **How AI actually works** — transformers, attention, MoE, diffusion, RLHF, test-time scaling
- 🔌 **The new power grid** — gigawatt training campuses, SMRs, dedicated nuclear, the energy build-out
- 🪛 **The chip wars** — TSMC 2nm, ASML EUV, NVIDIA Blackwell, Vera Rubin NVL72, US/China export controls
- 🏛️ **Geopolitics** — the comparative state of US, China, EU, UK frontier programs
- 🚧 **The data wall** — synthetic data, RLAIF, multimodal corpora, the post-data-scale playbook
- 🛡️ **Alignment & labor** — RLHF, constitutional AI, the labor-market shifts already underway
- 🎯 **AGI & ASI timelines** — what frontier-lab roadmaps actually say
- 🔭 **Beyond 2030** — quantum, neuromorphic, bio-AI

Every claim is **calibrated to mid-2026** primary-source developer releases, research papers, and benchmarks.

## ✨ Why this exists

The frontier of AI moves too fast for textbooks and too slow for X threads. There is no neutral, citation-driven, fast-updating reference that covers **labs + hardware + grid + geopolitics + alignment + roadmaps** in one place.

This is that reference.

## ▶️ Run locally

```bash
git clone https://github.com/iarhaaan/siliconeposh.git
cd siliconeposh
bun install            # or npm / pnpm install
bun dev                # http://localhost:3000
```

Build:

```bash
bun run build          # generates sitemap + production build
bun run preview
```

Tests:

```bash
bun test               # unit (vitest)
bun run test:e2e       # end-to-end (playwright)
```
